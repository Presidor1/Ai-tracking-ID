ℹ About This Project

ID Tracking AI (Kidnap/Banditry Geolocation Project) is an AI-powered platform that analyzes uploaded images to extract intelligence for identification, tracking, and potential geolocation.

✨ Key Features
  • 📷 Upload and analyze ID images
  • 🌍 Extract text, objects, and patterns
  • 🗺 Display results on interactive maps
  • 🤖 Powered by AI (OCR, Object Detection, ML)
  • 🔐 Secure and intended for authorized use only
⚠ Disclaimer: This project is developed strictly for research, safety, and authorized security purposes. It must not be misused for unlawful surveillance or harm.

Built with ❤️ using Flask, YOLOv8, Tesseract OCR, and Leaflet.js maps.
